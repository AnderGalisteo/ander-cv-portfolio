# Ander Galisteo CV and Portfolio

Public source repository for a Markdown-driven CV and portfolio website.

## Content

Website content lives in:

- `src/content/en`
- `src/content/es`

Repository documentation lives in `docs/` and is not part of the website.

## Commands

```bash
npm install
npm run validate
npm run dev
npm run test
```

## Deployment

Pushes to `main` build and deploy the website through GitHub Actions and GitHub Pages.
